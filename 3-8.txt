Ошибка: После ввода числа в буфере ввода остаётся символ новой строки (\n), 
а уже он сразу считывается вторым scanf("%[^\n]", str) как пустая строка.

Исправление: Добавляем пробел перед форматом во втором scanf, чтобы пропустить 
пробельные символы: scanf(" %[^\n]", str);

Исправленная программа:
#include <stdio.h>
int main()
{
    int a;
    scanf("%i", &a);
    printf("a = %i\n", a);

    char str[100];
    scanf(" %[^\n]", str);  // добавлен пробел перед %
    printf("str = %s\n", str);
}
